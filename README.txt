# Functionality of app includes the following: An Opening and Closing screen, a Main Menu screen, an Account Setup as well as a screen to display that info, a Travel screen, and a screen that displays all the cities traveled to.
# Follow the on-screen inscrucitons to use this application and to navigate through the Main Menu.
